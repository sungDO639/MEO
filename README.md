# MEO Code AI v2

AI-powered code assistant like Claude Code. Build, modify, and deploy full-stack apps with natural language.

## Quick Start

1. Clone the repo
2. Copy `.env.example` to `.env` and set your Anthropic API key
3. Run `docker-compose up -d`
4. Open http://localhost:3000

- AI chat & agent mode
- Full project understanding
- Code generation & safe modification
- Auto error detection and fixing
- Modern UI with Monaco editor, terminal, preview
